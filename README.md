# algoritmos-combinatorios-tarea-2

Repositorio creado para la tarea 2 del curso de algoritmos combinatorios por Andres Cruz Tello.

## Python project initialized

This repository now uses a standard Python `src` layout and `pyproject.toml`.

### Structure

- `pyproject.toml`: project metadata, packaging, and pytest config
- `src/algoritmos_combinatorios_tarea_2/`: main Python package
- `tests/`: test suite with `pytest`

### Quick start

```bash
python -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -e ".[dev]"
pytest
```
